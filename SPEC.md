# Robin Bobin — robinbobin.ru Specification

## Executive Summary

Одностраничный сайт кафе быстрого питания "Робин Бобин" (Челябинск) с каталогом меню и формой онлайн-заказа на самовывоз. Яркий, весёлый дизайн с GSAP-анимациями, фокус на быструю конверсию: увидел меню → оформил заявку.

## Problem Statement

У кафе нет веб-присутствия — только Telegram-бот. Сайт нужен для:
- Привлечения новых клиентов через поиск и рекламу
- Удобного просмотра меню с любого устройства
- Приёма заказов на самовывоз без звонков

## Success Criteria

- Сайт загружается < 2 сек (First Contentful Paint)
- Меню видно без скролла на мобильном (или в 1 скролл)
- Заявка отправляется в Telegram + Email за < 500ms
- Mobile-first, корректно на всех устройствах

## User Personas

**Основная аудитория:** широкая — от студентов до семей с детьми, Челябинск.
- Техническая грамотность: средняя (смартфон, мессенджеры)
- Цель: быстро выбрать еду, заказать, забрать

## User Journey

```
1. Открыл сайт → видит Hero с логотипом/маскотом и CTA "Смотреть меню"
2. Скроллит → секция Меню с категориями и карточками блюд
3. Выбирает блюда → добавляет в корзину (счётчик)
4. Нажимает "Оформить заказ" → модалка/форма (имя, телефон, точка самовывоза)
5. Отправляет → уведомление в Telegram администратору + Email
6. Видит подтверждение с номером заказа и временем ожидания
```

## Functional Requirements

### Must Have (P0)

- **Hero-секция**: логотип, маскот, слоган, CTA-кнопка "Смотреть меню"
- **Меню**: 4 категории (Бургеры, Шаурма, Закуски, Напитки), 15 позиций
  - Карточка: фото (сток), название, описание, цена
  - Фильтр по категориям (табы/кнопки)
  - Кнопка "Добавить" на каждой карточке
- **Корзина**: счётчик позиций, итоговая сумма, редактирование количества
- **Форма заказа**: имя, телефон, выбор точки самовывоза (2-3 адреса)
- **Уведомления**: отправка заявки в Telegram (ADMIN_CHAT_ID) + Email
- **Контакты**: адреса точек, карта (Яндекс), телефон, Telegram
- **Навигация**: sticky header с лого + якорные ссылки
- **Mobile-first**: адаптивность для всех экранов

### Should Have (P1)

- **О нас**: краткая история/философия кафе, фото интерьера
- **Отзывы**: карусель отзывов клиентов
- **GSAP-анимации**: плавные появления секций при скролле, parallax Hero
- **Lenis**: smooth scroll
- **Footer**: контакты, соцсети (Telegram), копирайт

### Nice to Have (P2)

- Выбор времени самовывоза
- Онлайн-оплата (заложить архитектурно)
- Промо-баннеры / акции
- SEO-оптимизация (мета-теги, schema.org для ресторана)

## Technical Architecture

### Stack

| Technology | Version | Purpose |
|-----------|---------|---------|
| React | 19+ | UI framework |
| Vite | 7+ | Build tool & dev server |
| Tailwind CSS | 3.4+ | Utility-first styling |
| GSAP + ScrollTrigger | 3.14+ | Scroll & entrance animations |
| Lenis | 1.3+ | Smooth scrolling |
| Puppeteer | latest | Screenshot/visual testing |

### Data Model

Меню — статический JSON (захардкожено, 15 позиций не требуют БД).

```
MenuItem {
  id: number
  name: string
  description: string
  price: number
  category: "burgers" | "shawarma" | "snacks" | "drinks"
  image: string (path)
  packaging?: string
}

Order {
  items: { menuItemId, quantity }[]
  customerName: string
  customerPhone: string
  pickupLocation: string (address)
  totalAmount: number
}
```

### System Components

```
[Browser] → [React SPA on Vite]
                ↓ (form submit)
        [API endpoint / n8n webhook]
                ↓
    ┌───────────┴───────────┐
    ▼                       ▼
[Telegram Bot API]    [Email (SMTP)]
```

**Backend для заявок**: n8n webhook или минимальный Express endpoint на том же сервере.

### Integrations

- **Telegram Bot API**: отправка заказа администратору (ADMIN_CHAT_ID: 183921686)
- **Email**: SMTP для дублирования заявки
- **Яндекс.Карты**: виджет карты с точками кафе

### Security Model

- Нет авторизации пользователей
- Rate limiting на endpoint заказа (защита от спама)
- Валидация телефона (формат)
- Санитизация input'ов

## Non-Functional Requirements

- **Performance**: FCP < 1.5s, TTI < 3s
- **Scalability**: статика + webhook, масштабирование не требуется
- **Reliability**: сайт работает всегда (статика), заказы — через n8n (уже стабильно)
- **SEO**: мета-теги, Open Graph, schema.org/Restaurant
- **A11y**: семантический HTML, alt-теги, focus-visible, контрастность

## Deployment

- Сервер: 217.114.10.31
- Reverse proxy: Traefik (file provider, как fastfood-bot)
- Домен: robinbobin.ru → Traefik → localhost:PORT (или статика)
- SSL: Let's Encrypt через Traefik
- CI/CD: ручной деплой (scp/rsync) или git pull

## Out of Scope

- Личный кабинет / регистрация пользователей
- Онлайн-оплата (v1)
- Доставка
- Админка для редактирования меню (меню статическое)
- Мультиязычность
- PWA / мобильное приложение
